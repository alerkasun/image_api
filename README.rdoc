# Resize Image
 - route index: GET www.domain.com/api/resize
 - format request: api/resize
```sh
-[HTTP_API_TOKEN] = 'Token iF9zRYsyDkihsAib_39H'
```
```sh
gallery[file]=file.jpg/png/gif/jpeg
gallery[size]='widthxheight'
```
